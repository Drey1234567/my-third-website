Git and Github

what is git

Git is an open source version control sysytem that allows developer track version of their code overtime.

Git track our code by creating snapshots of the code and can be revisited whenever nrccessary

Configuring Git

git config --global user.name "DARE" is used to configure our name on Git.

git config --global user.email "dreysingz77@gmail.com" is used to configure our email.

Basic Commands

git init is used to initialize git in your local repository

git statusis used to check the status of the activities of git in your local repository.

git add . is used to track a single file in git.

git rn --cached "filename" is used to untrack a file in git.

git commit -m "the commit message" is used when we want to save a snapshot of our code. it is like a caption or description.

git remote add origin master "repo URL is used to connect your local repo to github.

git push -u origin master is used to push our code rom the local repo to the remote repo,



What is Github

Github is a web based git repository hosting service

it is a tool that enables collaboration by hosting shared git repositories that team a developers

While Github uses git,the funvtional it provides is entirely different from git.

Git is a version control system while github is an online platform for hosting and sharing coe.

Why use Github
